# 🍅 番茄钟 - 项目需求规格文档

## 1. 项目概述

### 1.1 项目名称
Vibe Pomodoro Clock（氛围番茄钟）

### 1.2 项目简介
一款基于番茄工作法的时间管理 Web 应用，帮助用户专注工作、提升效率。

### 1.3 项目目标
- 提供简洁美观的番茄计时体验
- 帮助用户养成专注工作习惯
- 支持数据统计和持久化

## 3. 设计风格

### 3.1 整体风格
- **苹果风格（Apple Style）**：圆润边角、细腻阴影、毛玻璃效果
- **简约高级**：大量留白、精致细节、轻盈质感

### 3.2 色彩方案
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色调（工作） | 玫红 | `#f43f5e` (rose-500) |
| 主色调（休息） | 绿色 | `#10b981` (emerald-500) |
| 背景渐变 | 桃红渐变 | rose-200 → pink-200 → rose-300 |
| 卡片背景 | 半透明白 | white/30 + backdrop-blur-2xl |
| 文字主色 | 深玫红 | rose-600 / rose-700 |

### 3.3 字体
- 标题：`font-light`（细体）
- 计时器：`font-mono` 等宽字体
- 正文：`font-medium`（中等）

### 3.4 动效
- 按钮悬停：轻微上浮 + 阴影增强
- 状态切换：500ms 颜色渐变过渡
- 进度条：平滑线性动画
